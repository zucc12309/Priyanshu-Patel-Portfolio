CRM Workflow Automation

I built a workflow automation system for streamlining CRM export processing and reducing manual reporting effort.

Problem

CRM teams lose time to manual export cleanup, follow-ups, status checks, and reporting.

Solution

I used n8n, Python preprocessing, AI briefing generation, Power BI refresh, and success/error email paths to make the workflow auditable.

Architecture

Gmail trigger → n8n workflow → Python preprocessing → formatting script → AI briefing → Power BI refresh → success/error email.

Roadmap

I would expand this with credential hardening, more CRM export templates, SLA monitoring, and broader dashboard coverage.
